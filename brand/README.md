# 누가일등 브랜드 자산

## 파일
- logo-symbol.svg           심벌 단독 (밝은 배경용)
- logo-horizontal.svg       심벌+워드마크 가로형 (밝은 배경)
- logo-horizontal-white.svg 가로형 화이트 (앱바·전광판 등 어두운 배경)
- app-icon.svg / app-icon-512.png   앱 아이콘 원본·512px
- apple-touch-icon-180.png  iOS 홈 화면 추가용
- favicon.svg / favicon-32.png      브라우저 탭 아이콘

## 컬러
- 메인 블루   #4E8FD0
- 딥 블루     #2F6BAA (아이콘 배경, 앱바)
- 하이라이트  #F7CF5C (1등 줄 전용 — 남용 금지)
- 잉크        #22303D

## 사용 규칙
- 옐로우는 "1등/내 것" 강조에만. 두 번째 강조색이 필요해도 옐로우를 늘리지 말 것.
- 워드마크 서체: Noto Sans KR 900 (자간 -0.01em). 텍스트를 이미지 위에 얹을 때는 화이트 버전 사용.
- 심벌 최소 사용 크기 16px. 그 이하에서는 favicon 단순화 버전을 쓸 것.
