# 누가일등 브랜드 v3 — 말풍선 로고 (2026.07 확정)

## 컨셉
"누가 일등이야?" — 브랜드명이 곧 질문. 말풍선 속에서 '일'이 황금 숫자 1로 빛난다.
결정 이력: v1 리더보드 심벌 → v2 배지형 → v3 말풍선(E2안, 컬러 C: 딥네이비×화이트×골드) 확정.
검토 후 기각: 색종이 점 장식(과함), 900 굵기(비대), 골드 풍선 대표안(흰 지면 대비 약함 — 반전판으로만 사용).

## 파일
- logo-main.svg / logo-main-2000.png   ★대표 (딥네이비 #1D3D5C 풍선 · 화이트 워드마크 700 · 골드 #F7CF5C 1)
- logo-light.svg / logo-light-2000.png  라이트판 (화이트 풍선+딥블루 테두리 · 잉크 텍스트 · 골드 1) — 어두운 지면/인쇄용
- app-icon.svg / app-icon-512.png / apple-touch-icon-180.png  1-풍선 아이콘 (같은 숫자 도형 공유)
- favicon.svg / favicon-32.png
- logo-appbar.svg / logo-appbar-800.png  앱바 전용 무풍선 변형 (흰 워드마크+골드 1, 투명 배경) — 딥블루 앱바 위에서 풍선 대비가 죽는 문제의 해법

## 컬러 정합 (v3 확정)
- UI 팔레트 전면 유지 (블루 메인 확정 이력 유효). 네이비 #1D3D5C를 --navy 정식 토큰으로 승격: 용처 = 로고·LIVE 카드·전광판·다크 표면
- 로고의 골드 1 = UI 옐로우(#F7CF5C)와 동일 — "옐로우=1등" 규칙의 상징

## 사용 규칙
- 밝은 지면 → logo-main / 어두운 지면·단색 인쇄 → logo-light
- 작은 지면(아이콘·파비콘) → 1-풍선. 워드마크 풍선을 40px 이하로 줄이지 말 것
- 워드마크 굵기 700 고정(로고 전용 규격). 숫자 1은 항상 골드(#F7CF5C, 라이트판 #E8B33A)
- 풍선·꼬리는 단일 패스 — 분리·변형 금지. UI 컬러 규칙(옐로우=1등)과 공존: 로고의 골드 1이 곧 그 규칙의 상징
